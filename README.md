# Simple Banking System (C++)

A console-based banking system built in C++.

## Features
- Create accounts with PIN protection
- Deposit, Withdraw, Transfer money
- Transaction history (last 5 per account)
- Admin panel with bank summary
- Data saved to file (survives restart)

## How to Run
```
g++ -std=c++17 -o banking main.cpp
./banking
```
